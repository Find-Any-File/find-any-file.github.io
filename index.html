Find Any File

Search your local and network disks for files using precise criteria: name, creation or modification date, size, or kind. Unlike Spotlight, it scans volumes directly—uncovering files inside packages, system files, and anything hidden from standard searches.

Hierarchical Results View

Browse hundreds of results effortlessly in a new tree-like structure. Navigate folders and files intuitively, without the clutter of flat lists.

Root-Level Access

Run as root to find truly any file on your disk—even those invisible to normal users. Perfect for system files, protected directories, and hidden content.

Save and Reuse Queries

Capture complex searches as reusable queries. Run them anytime with one click, streamlining your workflow for recurring tasks.

Direct Volume Scanning

Bypasses Spotlight's database for exhaustive searches on HFS+ volumes and network drives. Fast, reliable, and unfiltered—ideal when Spotlight falls short.

How It Works

Step 1: Define Your Criteria

Select disks, then set filters for name, dates, size, or file kind. Build precise queries tailored to your needs.

Step 2: Scan Directly

Launch the search—it indexes volumes live, revealing files Spotlight misses, including packages and exclusions.

Step 3: Explore Hierarchically

Results appear in a navigable tree view. Drill down through folders to pinpoint files instantly.

Step 4: Save or Act

Save the query for later, or open, copy, or reveal files directly in Finder.

Frequently Asked Questions

Is it faster than Spotlight?

It scans directly, so it's thorough but may take longer on large volumes. Optimized for HFS+ and surprisingly quick on network shares.

Does it find hidden system files?

Yes—with root access, it uncovers everything, even files protected from standard users.

Can I reuse searches?

Absolutely. Save queries and rerun them effortlessly.
